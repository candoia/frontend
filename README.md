# candoia-frontend

### Building

1. [Download and install iojs](https://iojs.org/).
2. Clone the frontend repository (`git clone ...`)
3. `cd` to the cloned repository
4. npm install electron: `npm install electron-prebuilt --save-dev`
5. run `npm install`
6. To start the program, type `npm start` while in the repository directory



### Project Layout

- `app` - Candoia goes here
- `config` - place for environment specific stuff
- `build` - in this folder lands built, runnable application
- `releases` - ready for distribution installers will land here
- `resources` - resources for particular operating system
- `tasks` - build and development environment scripts
