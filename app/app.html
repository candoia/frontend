Getting Started with Candoia

    Add a repository

    Start by adding your own git repository on the left-hand side, or use one of the existing ones.

    Run an app

    Right click on one of the repositories and choose an app to run. Depending on the size of the repository and the application chosen, it might take a few seconds.

    Install a new app

    Install new apps by clicking Install App on the bottom panel.
